# CLIB

因为在Windows平台开发一个项目，需要一个数据结构操作的库，但是看了下别人写的东西，有点看不懂
所以，就自己写一个这样的库给自己用。之所以看不懂是由于本人是各种语言都写一点，但不陷进去的那
种程序员，水平有限，力争用最简洁的语法做完这个功能模块。由于我比较喜欢C这门编程语言，所以会
比较享受这个过程。

C语言是个非常重要的编程语言，在一些应用开发中，如果不熟练的掌握C语言指针操作，基本无法高效率
的做任何事，相信我，如果学计算机，C语言是你无法绕过的坎，如果不精通C语言，那也就永远只是个脚
本小子，大菜逼，做不了什么很hack的事。严格来说，不会C，都不算什么程序员。那些做网页的，搞java
应用的，他们算个鸟啊，狗鸡巴一样的自以为是的蠢狗。

这次要做的事情有这么几个：

[]. 数组实现列表
[]. 链式实现列表
[]. 数组实现队列
[]. 链表实现队列
[]. 数组实现栈
[]. 链表实现栈
[]. 基于数组实现哈希表
[]. 基于链表实现哈斯表

上面这些组件，需要支持线程安全和非线程安全的切换。同时，还希望能够支持跨平台。
