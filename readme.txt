# CLIB

因为在Windows平台开发一个项目，需要一个数据结构操作的库，但是看了下别人写的东西，有点看不懂
所以，就自己写一个这样的库给自己用。之所以看不懂是由于本人是各种语言都写一点，但不陷进去的那
种程序员，水平有限，力争用最简洁的语法做完这个功能模块。由于我比较喜欢C这门编程语言，所以会
比较享受这个过程。

这次要做的事情有这么几个：

[]. 数组实现列表
[]. 链式实现列表
[]. 数组实现队列
[]. 链表实现队列
[]. 数组实现栈
[]. 链表实现栈
[]. 基于数组实现哈希表
[]. 基于链表实现哈斯表

上面这些组件，需要支持线程安全和非线程安全的切换。同时，还希望能够支持跨平台。
